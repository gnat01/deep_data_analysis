# Source Layout

This directory holds the executable Python modules for the YC hiring-posts project.

Current modules:

- `cli.py`
- `discovery.py`
- `fetch.py`
- `models.py`
- `parse.py`
- `source_index.py`
- `storage.py`
- `validate.py`

The layout is intentionally flat for now. If the codebase grows substantially, it can be split into subpackages later.
